1. Установить Node.js

Скачать и установить:
https://nodejs.org

Проверка:

node -v
npm -v
2. Склонировать проект
git clone https://github.com/Vanusha61/hakaton.git
cd hakaton
3. Перейти во frontend
cd frontend
4. Установить зависимости
npm install
5. Запустить проект
npm run dev
6. Открыть в браузере
http://localhost:5174/
