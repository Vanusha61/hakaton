# Реализовать следующие эндпониты

## Количество уроков, в которых пользователь решил все задания

Нужно вывести данные по четвертям и по году - количество уроков, в котором пользователь решил все задания

### Формат ответа для четверти

```json
{
  "user_id": <int>,
  "course_id": <int>,
  "quarted_id": <int>, # 1, 2, 3, 4
  "full_solved_lessons_count": <int>, # Количество уроков, в которых пользователь решил все задания
}
```

### Формат ответа для года

```json
{
  "user_id": <int>,
  "course_id": <int>,
  "full_solved_lessons_count": <int>, # Количество уроков, в которых пользователь решил все задания
}
```

## Не прогулял ли пользователь ни одного урока

Эндпоинт должен отвечать на вопрос не прогулял ли пользователь ни одного урока за четверть и за год

Ответ должен быть таким:

### Четверть

```json
{
  "user_id": <int>,
  "course_id": <int>,
  "quarted_id": <int>, # 1, 2, 3, 4
  "is_user_visited_all_lessons": <bool>, # Посетил ли пользователь все уроки
}
```

### Год

```json
{
  "user_id": <int>,
  "course_id": <int>,
  "is_user_visited_all_lessons": <bool>, # Посетил ли пользователь все уроки
}
```

## Топ 3 сложных урока

Вводим Топ 3 урока, задания в котором, пользователь решил, но не решили или мало решали другие студенты  И дополнитьно выводим процент пользователей к каждому уроку, которые не справились с заданием  Еее решили 5 % поьзователей

Например,
1. Урок: «Указатели на C++», ты решил, а не справились 85% учеников
2. и так далее

Также по четвертям и годам

### Четверть

```json
{
  "user_id": <int>,
  "course_id": <int>,
  "quarted_id": <int>, # 1, 2, 3, 4
  "rare_lessons": {

    },
    "2": {
      "title": <str>, # название урока, который решили меньше всего учеников (топ-2), а user_id - решил,

  },
```
